# Лабораторна робота №1: Web-дизайн

**Студентка:** Чучвага Анастасія
**Група:** КВ-32
**Тематика додатку:** Опитування (SurveyMaker)

## Назва роботи
Розробка статичного інтерфейсу Web-додатка.

## Завдання до роботи
Розробити статичні Web-сторінки додатку та перевірити коректне їх відображення на екранах різного розміру. Ознайомитись із засобами мов HTML5 та CSS3, навчитись застосовувати бібліотеку Twitter Bootstrap для швидкої побудови Web-інтерфейсу користувача.

## Структура проєкту
* `index.html` - Головна сторінка (створення опитувань).
* `register.html` - Сторінка реєстрації користувача.
* `login.html` - Сторінка входу до сайту.
* `profile.html` - Профіль користувача.
* `about.html` - Інформація про додаток.
