# Лабораторна робота 1 та 2 з дисципліни Web-дизайн

Студентка: Чучвага Анастасія
Група: КВ-32
Тематика додатку: Опитування (SurveyMaker)

## Опис проєкту
Цей репозиторій містить розробку Web-додатка для створення та проходження опитувань. Проєкт реалізовано поетапно в рамках виконання лабораторних робіт.

## Лабораторна робота 1: Статичний Web-дизайн
Завдання: Розробити статичні Web-сторінки додатку та перевірити коректне їх відображення на екранах різного розміру. Ознайомитись із засобами мов HTML5 та CSS3, навчитись застосовувати бібліотеку Twitter Bootstrap для швидкої побудови Web-інтерфейсу користувача.

## Лабораторна робота 2: Розробка функціональності Web-додатка мовою Javascript
Завдання: Розробити функціональність для статичних сторінок Web-додатку першої лабораторної роботи із використанням шаблону MVC (Model-View-Controller). Web-фреймворки (ReactJS, AngularJS, VueJS) не використовувати. Інструменти розробки: Vanilla JavaScript.

Основний реалізований функціонал:
* Уся логіка розділена на ізольовані класи згідно з патерном MVC.
* На головній сторінці реалізовано динамічне додавання та видалення полів для варіантів відповідей.
* Налаштовано збереження створених опитувань у пам'ять браузера за допомогою LocalStorage.
* Додано можливість реєстрації та входу до системи із записом даних у локальне сховище.
* На сторінці профілю реалізовано автоматичне підтягування персональних даних активного користувача.

## Структура проєкту
* index.html - Головна сторінка з підключеною логікою створення опитувань.
* register.html - Сторінка реєстрації користувача із записом даних.
* login.html - Сторінка входу до сайту із перевіркою паролів.
* profile.html - Профіль користувача із динамічним відображенням параметрів.
* about.html - Інформація про додаток.
